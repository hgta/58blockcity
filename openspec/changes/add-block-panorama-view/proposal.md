# 区块城市九区全景视图 (L1)

## 背景

block.58.tl 的区块城市页面（`city.php`）目前通过 Tab 切换展示单个区域（A-H, Z），一次只能看一个区的 101×99 网格。blockcity.vip 原站也是各区完全独立。

## 目标

新增"全城全景"模式，在一个页面内同时展示 9 个区域的缩略图矩阵（3×3 排列），作为 block.58.tl 区别于原站的核心视觉亮点。

## 方案

- `city.php` 默认显示"全城全景"模式
- 顶部 Tab 栏：`[🏙️ 全城] [A区] [B区] ... [Z区]`
- 全景模式：9 个区以 3×3 缩略图排列，每格 3px，每区 303×297px
- 点击任意区 → 切到该区详细 101×99 网格（保留原功能）
- 每区卡片显示：区名、已售数量、占比、迷你热力图

## 范围

- 修改 `block/city.php`：增加全景渲染模式
- 可能微调 `classes/Block.php`：增加统计方法

## 不包含

- 跨区合并（L2）
- BCT 支付打通（L3）
