# 视频截帧作为商品主图

## 问题
新增商品时只上传视频不上传主图，会提示"请上传商品主图"。视频在上传后应支持截取某一帧作为封面图。

## 规则
1. 有主图文件 → 优先使用（不变）
2. 只上传视频 + 从视频截取封面 → 自动生成主图+缩略图
3. 只上传视频但未截封面 → 提示"请上传主图或从视频截取封面"
4. 都未传 → 提示"请上传商品主图"

## 技术方案
- **客户端 Canvas 截图**（FFmpeg 不可用）
- JS `<video>` + `<canvas>` 捕获帧 → base64 → POST 到后端
- PHP `base64_decode` + GD 缩放 → 保存为 jpg 主图+缩略图
