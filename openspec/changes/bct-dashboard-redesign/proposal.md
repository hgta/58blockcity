# BCT 用户 Dashboard 重新设计

## 背景

`bct/user/dashboard.php` 混合了 BCT 交易和互访圈内容，布局不合理，缺少订单有效期和取消删除功能。

## 问题

1. 右侧 1/3 全是互访圈内容（互访圈列表、待处理访问、访问记录），与 BCT 无关
2. 统计卡片含互访圈数据（互访圈数、待处理访问）
3. 订单无有效期 → 挂单永久存在，无法过期清理
4. 订单无取消/删除 → 挂错了只能放着
5. 冻结余额机制复杂且无用 → 下单时直接扣余额更简单

## 目标

- 移除所有互访圈相关内容，纯 BCT 交易面板
- 订单支持有效期（7/15/30/60/90天/永久）
- 支持取消/删除订单
- 简化余额管理，去掉冻结
- 优化布局和统计

## 范围

### 包含
- DB: `bct_orders` 加 `expires_at` 列
- `BCTOrder`: `createOrder` 加 duration，新增 `cancelOrder`，去掉冻结
- `dashboard.php`: 全页重写

### 不包含
- 互访圈功能本身修改（只是仪表盘不再引用）
- 交易匹配逻辑修改
- transfer.php 功能修改
