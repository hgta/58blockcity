# Design: 全站移动端响应式优化

## 1. shared/header.php — 汉堡菜单

### 当前

```html
<header>
  <div class="header-container">
    <div class="logo">...</div>
    <div class="header-actions">
      <!-- 7-8个 nav-button，768px 以下只是缩小+wrap -->
    </div>
  </div>
</header>
```

### 方案

纯 CSS + 少量 JS 实现，无需引入额外库：

```
桌面 (>768px):
┌──────────────────────────────────────────────┐
│ [58] 人气值    首页 商品 排行 ...   登录 注册  │
└──────────────────────────────────────────────┘

手机 (≤768px):
┌────────────────────────────────┬────────────┐
│ [58] 人气值                     │    ☰       │
└────────────────────────────────┴────────────┘
         ▼ 点击展开（全屏下拉菜单）
┌──────────────────────────────────────────────┐
│  🏠 首页                                     │
│  🛍 商品浏览                                 │
│  🏆 排行榜                                   │
│  🏪 店铺列表                                 │
│  🛒 购物车                                   │
│  📋 我的订单                                 │
│  ─────────────────────────────              │
│  👤 个人中心                                 │
│  🚪 退出                                     │
└──────────────────────────────────────────────┘
```

**技术方案：**
- 新增 `.menu-toggle` 按钮，默认隐藏（`display:none`）
- `@media(max-width:768px)` 显示按钮、隐藏 `.header-actions`
- 点击 toggle 切换 `.header-actions.open` 展开为全宽下拉列表
- 展开时 `.nav-button` 变为块级元素，左边对齐，间距加大（触控友好）
- 点击菜单外部或再点 toggle 关闭

## 2. payment-settings.php — 表格移动端

### 当前

```
| 城市 | 接收区块 | 启用 |     ← 3列表格
| 北京 | [select] | [✓] |
| 上海 | [select] | [✓] |
| ...50行...
```

### 方案

`@media(max-width:768px)` 三列合并为卡片式：

```
┌─────────────────────────────┐
│ 北京                        │
│ 接收区块: [select dropdown] │
│ 启用: [✓]                   │
├─────────────────────────────┤
│ 上海                        │
│ 接收区块: [select dropdown] │
│ 启用: [✓]                   │
└─────────────────────────────┘
```

实现：表格 thead 隐藏，td 设 `display:block`，用 `::before` 伪元素加标签。

## 3. products.php — 工具栏适配

### 当前

```
[全选] [批量上架] [批量下架] [批量删除]    [全部|在售|下架|草稿] [搜索框]
```

### 方案

`@media(max-width:768px)` 堆叠为两行：
- 第一行：筛选 tabs（全部/在售/下架/草稿）
- 第二行：搜索框 + 批量操作

`@media(max-width:480px)` 三行：
- 筛选 tabs 可横向滚动
- 搜索框全宽
- 批量操作按钮缩小

## 4. 表单页通用优化 — create.php / profile.php / security.php

### 方案

添加 `@media(max-width:768px)` 基础规则：
- `.form-group` 内的 `input/select/textarea` 改为 `width:100%`
- `.row` 内的 `.col-*` 变为 `width:100%; max-width:100%`
- 按钮 `min-height:44px`（符合 iOS 触控建议）
- 提交按钮 `width:100%`
