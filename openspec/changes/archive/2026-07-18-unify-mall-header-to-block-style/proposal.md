# 统一 Mall 头部样式为 Block 风格

## 背景

`block.58.tl` 和 `mall.58.tl` 是同一体系下的两个子站，但目前头部样式差异巨大：

| 维度 | Block（目标） | Mall（当前） |
|------|:-----------:|:-----------:|
| 头部背景 | 白色 `#fff` | 橙色 `#ff6b00` |
| 分隔方式 | `border-bottom: 1px solid` | `box-shadow` |
| 头部高度 | 固定 56px | 自适应 padding |
| Logo 图标 | 橙色渐变底+白字 34px | 白色底+橙色字 44px |
| 导航按钮形状 | 6px 圆角 | 20px 胶囊形 |
| 导航文字颜色 | 深灰 `#666` | 白色 |
| 导航 hover | 浅橙底+橙色字 | 半透明白底 |
| 注册按钮 | 实心橙色 CTA | 透明，无突出 |

Block 的设计更简洁、专业，且与整体品牌视觉一致。统一 Mall 头部可提升跨子站体验一致性。

## 目标

将 `shared/header.php` 中 Mall 站的头部样式完全对齐 Block 站风格。

## 范围

### 纳入范围

- `shared/header.php` — 修改头部 CSS（背景、logo、导航、Hover 等）
- `shared/header.php` — HTML 结构调整（logo 图标、注册按钮样式）
- Mall 的城市定位条保留（Block 站没有，但 Mall 站有独立需求）

### 不纳入范围

- Block 站头部不改动
- Mall 站页面内容区不改动
- 其他子站（bct/hufang/nft）不改动

## 成功标准

| 指标 | 当前 | 目标 |
|------|------|------|
| 头部背景 | 橙色 | 白色（与 Block 一致） |
| 导航文字 | 白色 | 深灰 `#666` |
| Logo 图标风格 | 白色底 | 橙色渐变底+白字 |
| 注册按钮 | 透明胶囊 | 实心橙色 CTA |
| 跨子站视觉一致性 | 明显不同 | 统一风格 |
