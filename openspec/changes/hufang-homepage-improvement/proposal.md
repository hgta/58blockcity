# 互访圈首页改进

## 问题
1. 280个城市标签全部渲染，页面臃肿
2. 单城市最多60个圈子，无分页，无"查看更多"
3. 只有卡片模式，无列表模式
4. 无访问状态标记，不知道哪些圈子已访问过
5. 无搜索功能

## 改进
1. 城市标签折叠：默认显示热门20个 + "展开全部"
2. 卡片/列表双模式切换（URL参数 `?view=list`）
3. 单城市最多显示100个 + "查看全部"链接到分页页 `circles/all.php?city=xxx`
4. 已访/已互访水印标记（基于 visits 表 status）
5. 加入搜索框

## 涉及文件
| 文件 | 改动 |
|------|------|
| `hufang/index.php` | 主要改动 |
| `hufang/circles/all.php` | 新建分页页 |
| `classes/Circle.php` | 新增分页+计数方法 |
| `classes/Visit.php` | 新增批量获取已访圈子ID |
