# 增加用户粘性功能

## 背景

全站跳出率高，5个核心详情页是"死胡同"——看完内容直接进入footer，无任何引导。店铺创建串号事故后流失的用户需要重新吸引。

## 目标

为死胡同页面增加推荐引导，全站加回到顶部，减少跳出。

## 核心改造

| # | 页面 | 增加内容 |
|---|------|------|
| 1 | 全站 shared/footer | 回到顶部浮动按钮 |
| 2 | block/block/view.php | 同城市其他区块推荐 |
| 3 | nft/nft/view.php | 同标签NFT推荐 |
| 4 | mall/shop/view.php | 本店热门商品 |
| 5 | mall/product/detail.php | 增强推荐：店铺其他+看了又看 |
